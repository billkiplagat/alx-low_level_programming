# **SINGLY LINKED LIST PROJECTS**
This repository includes various projects related to singly linked list operations in C programming language. The projects include:

* Printing all elements in the singly linked list
* Adding an element to the start of the singly linked list
* Adding an element to the end of the singly linked list
* Deleting an element at the nth index of the singly linked list
* Inserting an element at the nth index of the singly linked list
* Deleting the first element in the singly linked list
* Freeing the entire singly linked list
